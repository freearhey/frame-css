Usage

To use the framework, simply add a link to it in the <head> tag:

<html>
  <head>
    ...
    <link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/freearhey/frame-css/dist/frame-css.min.css">
  </head>
  <body>
    ...
  </body>
</html>
